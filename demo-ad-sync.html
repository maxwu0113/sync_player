How It Works

  • Automatic Detection: The extension detects when YouTube ads are playing using multiple DOM indicators
  • Real-time Sync: Ad state is synchronized between all users in the room every second
  • User Notification: When someone else is watching an ad, you see the notification above and your video pauses
  • Automatic Resume: When everyone finishes watching ads, playback resumes automatically
